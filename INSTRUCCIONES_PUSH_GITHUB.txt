## 🆘 SOLUCIÓN DE PROBLEMAS

### "Repository not found"

```bash
# Verificar remote
git remote -v

# Agregar si no existe
git remote add origin https://github.com/[usuario]/academia_de_belleza.git
```

### "Permission denied (publickey)"

```bash
# Configurar SSH key
ssh-keygen -t rsa -b 4096 -C "tu-email@example.com"
# Agregar el key a GitHub Settings → SSH Keys
```

### "Conflictos al hacer push"

```bash
# Traer cambios
git pull origin Brigith

# Resolver conflictos manualmente

# Continuar
git add .
git commit -m "resolve conflicts"
git push origin Brigith
```
