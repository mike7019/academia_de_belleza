## 🔄 FLUJOS IMPLEMENTADOS

### CREAR MATRÍCULA (US 115)

```
[Nueva Matrícula] → Dialog
    ├─ Seleccionar Estudiante (activo)
    ├─ Seleccionar Curso (ABIERTO, muestra cupos)
    ├─ Ingresa Descuento (opcional)
    └─ [Guardar]
        ├─ ✓ Valida estudiante activo
        ├─ ✓ Valida curso ABIERTO
        ├─ ✓ Valida cupo disponible
        ├─ ✓ Valida sin duplicados
        ├─ ✓ Calcula valor_final = precio - descuento
        ├─ ✓ Transacción INSERT
        └─ ✓ Notificación "Creada correctamente"
```

### CANCELAR MATRÍCULA (US 116)

```
[Seleccionar en tabla] → [Cancelar Matrícula]
    ├─ Valida: no CANCELADA
    ├─ Valida: no FINALIZADA
    └─ [Dialog Confirmación]
        ├─ Muestra: Estudiante, Curso, Estado actual
        ├─ Muestra: "Se liberará cupo"
        └─ [OK]
            ├─ Cambia estado → CANCELADA
            ├─ Transacción UPDATE
            ├─ Libera cupo automáticamente
            └─ Notificación "Cancelada. Cupo liberado"
```

## 🗄️ ESQUEMA BD USADO

```
matricula (tabla)
├─ id: BIGSERIAL (PK)
├─ fecha: DATE
├─ estado: VARCHAR(20) [PENDIENTE|ACTIVA|CANCELADA|FINALIZADA]
├─ valor_base: NUMERIC(12,2)
├─ descuento: NUMERIC(12,2)
├─ valor_final: NUMERIC(12,2)
├─ observaciones: TEXT
├─ estudiante_id: BIGINT (FK → estudiante)
├─ curso_id: BIGINT (FK → curso)
└─ UNIQUE (estudiante_id, curso_id)

curso (tabla)
├─ id: BIGSERIAL (PK)
├─ estado: VARCHAR(20) [PLANIFICADO|ABIERTO|CERRADO|CANCELADO]
└─ cupo_maximo: INT

estudiante (tabla)
├─ id: BIGSERIAL (PK)
└─ activo: BOOLEAN
```
