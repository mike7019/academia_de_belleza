📋 DESCRIPCIÓN:
  Como usuario, quiero visualizar y filtrar matrículas para consultar información
  de inscripciones.

✅ CRITERIOS CUMPLIDOS:
  ✅ Se listan todas las matrículas
  ✅ Filtros por ESTADO (Todos, PENDIENTE, ACTIVA, CANCELADA, FINALIZADA)
  ✅ Filtros por ESTUDIANTE (Todos + estudiantes activos)
  ✅ Filtros por CURSO (Todos + todos los cursos)
  ✅ Se muestra ESTADO de cada matrícula
  ✅ Se muestra VALORES (base, descuento, final)
  ✅ Se muestra información del ESTUDIANTE (nombre, documento)
  ✅ Se muestra información del CURSO (nombre)

════════════════════════════════════════════════════════════════════════════════

📊 TABLA DE MATRÍCULAS RESULTANTE:

Columnas (8):
  1. Estudiante      → Nombre completo
  2. Documento       → Número de documento
  3. Curso           → Nombre del curso
  4. Fecha           → Fecha de matrícula
  5. Estado          → PENDIENTE/ACTIVA/CANCELADA/FINALIZADA
  6. Valor Base      → Precio del curso
  7. Descuento       → Monto descontado
  8. Valor Final     → Base - Descuento

Paginación: 15 registros por página

════════════════════════════════════════════════════════════════════════════════

🔄 FILTROS DISPONIBLES:

1. ESTADO: Todos | PENDIENTE | ACTIVA | CANCELADA | FINALIZADA
   → Default: "Todos"
   → Efecto: Filtra solo matrículas con ese estado

2. ESTUDIANTE: Todos | [Estudiantes activos]
   → Default: null (Todos)
   → Efecto: Filtra solo matrículas del estudiante seleccionado

3. CURSO: Todos | [Todos los cursos]
   → Default: null (Todos)
   → Efecto: Filtra solo matrículas del curso seleccionado

Botón LIMPIAR:
  → Reset todos los combos a valor inicial
  → Recarga tabla con todas las matrículas

════════════════════════════════════════════════════════════════════════════════

🎨 EJEMPLOS DE USO:

Caso 1: Ver todas las matrículas
  Estado: Todos | Estudiante: Todos | Curso: Todos
  → Tabla: TODAS las matrículas

Caso 2: Ver matrículas de un estudiante
  Estado: Todos | Estudiante: Juan Pérez | Curso: Todos
  → Tabla: Matrículas de Juan en todos los cursos (cualquier estado)

Caso 3: Ver matrículas de un curso
  Estado: Todos | Estudiante: Todos | Curso: Corte Básico
  → Tabla: Matrículas en Corte Básico (todos los estudiantes)

Caso 4: Ver matrículas activas
  Estado: ACTIVA | Estudiante: Todos | Curso: Todos
  → Tabla: Solo matrículas en estado ACTIVA

Caso 5: Búsqueda específica
  Estado: ACTIVA | Estudiante: Juan | Curso: Corte
  → Tabla: SOLO matrículas de Juan ACTIVA en Corte

Caso 6: Limpiar
  [Limpiar] → Todos vuelven a inicio → Todas las matrículas
