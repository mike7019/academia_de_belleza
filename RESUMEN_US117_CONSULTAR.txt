                    ✅ USER STORY 117 - CONSULTAR MATRÍCULAS
═══════════════════════════════════════════════════════════════════════════════

📋 DESCRIPCIÓN
──────────────
Como usuario, quiero visualizar y filtrar matrículas para consultar información
de inscripciones.

✅ CRITERIOS CUMPLIDOS
────────────────────────
✅ Se listan todas las matrículas
✅ Filtros por ESTADO (Todos, PENDIENTE, ACTIVA, CANCELADA, FINALIZADA)
✅ Filtros por ESTUDIANTE (Todos + estudiantes activos)
✅ Filtros por CURSO (Todos + todos los cursos)
✅ Se muestra ESTADO de cada matrícula
✅ Se muestra VALORES (base, descuento, final)
✅ Se muestra ESTUDIANTE (nombre, documento)
✅ Se muestra CURSO (nombre)

💾 FLUJOS DE FILTRO
──────────────────

FLUJO 1: Ver todas las matrículas
  Estado: Todos ▼
  Estudiante: Todos ▼
  Curso: Todos ▼
  → Tabla muestra TODAS las matrículas (15 x página)

FLUJO 2: Filtrar por estudiante
  Estado: Todos ▼
  Estudiante: [Juan Pérez ▼]
  Curso: Todos ▼
  → Tabla: solo matrículas de Juan (ACTIVA + CANCELADA + etc)

FLUJO 3: Filtrar por curso
  Estado: Todos ▼
  Estudiante: Todos ▼
  Curso: [Corte Básico ▼]
  → Tabla: solo matrículas del curso (todos los estudiantes)

FLUJO 4: Filtrar por estado
  Estado: [ACTIVA ▼]
  Estudiante: Todos ▼
  Curso: Todos ▼
  → Tabla: solo matrículas ACTIVA (todos los estudiantes + cursos)

FLUJO 5: Combinación
  Estado: [ACTIVA ▼]
  Estudiante: [Juan Pérez ▼]
  Curso: [Corte Básico ▼]
  → Tabla: SOLO matrículas de Juan ACTIVA en Corte Básico

FLUJO 6: Limpiar
  [Limpiar]
  → Todos vuelven a "Todos"
  → Tabla refresca con todas

   • StringConverter: muestra nombre (no ID)

   • 15 registros por página
   • Navegación con botones numerados

✅ Visualización clara
   • 8 columnas con información relevante
   • Valores en formato moneda (12,2 decimales)

✅ Manejo de errores
   • Validación: Permiso en service
   • UI: Notificaciones visuales
